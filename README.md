# 💰 Expense Tracker

A PHP-based Expense Tracker developed as a Minor 1 project.  
This application helps users record, manage, and analyze their daily expenses in a simple and efficient way.

---

## 📌 About

Managing daily expenses is an essential part of financial planning.  
This project was built to provide a simple solution for tracking income and expenses while applying core web development concepts.

The system allows users to store financial records, monitor spending, and maintain better control over their budget.  
It simulates a real-world personal finance management system.

---

## 🎯 Objective

- To build a practical web-based expense management system  
- To apply PHP and backend development concepts  
- To understand database integration and CRUD operations  
- To improve full-stack development skills  

---

## 🚀 Features

- Add new expenses with details  
- View all recorded expenses  
- Update existing expense records  
- Delete expenses

## ⚙️ How It Works

1. User opens the application in a browser  
2. The system connects to the database  
3. User can:
   - Add new expenses  
   - View all expenses  
   - Update existing records  
   - Delete unwanted entries  
4. All data is stored in a database  
5. The system retrieves and displays expense summaries

## ▶️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/PramishBytes/expense_tracker.git
```

### 2. Move to Server Directory

- Place the folder inside `htdocs` (for XAMPP) or `www` (for WAMP)

### 3. Setup Database

- Open phpMyAdmin  
- Create a new database  
- Import the `.sql` file from the `database/` folder  

### 4. Run the Project

- Start Apache and MySQL  
- Open browser and go to:

```
http://localhost/expense_tracker
```
